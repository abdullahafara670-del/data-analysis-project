# data-analysis-project 

# Overview
This project explores stock market data through exploratory data analysis (EDA) and basic predictive modeling.

## Tools Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
Power BI
Jupyter Notebook

## Key Insights
Identified overall upward and downward trends in stock price movements over time.
Discovered that market volatility increases during high trading volume periods.
Found strong correlation between closing price and moving averages, useful for trend prediction.

## Project Goals
Clean and analyze stock market data
Perform exploratory data analysis (EDA)
Visualize insights using Power BI
Build a simple predictive understanding of stock movements

## Project Outcome
This project improves understanding of:
Financial data analysis
Time-series patterns
Data visualization techniques
Real-world application of Python and Power BI
